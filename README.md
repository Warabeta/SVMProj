# 中科院科学训练营
## 步态分析app
###一个简单的利用SVM算法实现的训练一个人 走路，跑步，静止，以及手机位置等不同情况下的特征，进行模型训练之后进行分类。

 **一些细节介绍**

 ***利用手机自带传感器进行数据收集。***  
 
 ***核心算法为SVM算法。***    
 
 ***一共设置了9中运动方式如走路，静止，还设置了20种位置方式如手持，口袋。***
